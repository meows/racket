Undoing what you did

If you had an integer x, and I added 3 to it to get a new number y, could you reverse what I did and get back your original number? For example:

;; Somewhere above I secretly defined x -- but I won't let you see the number!
;; Now you must fine out how to get back the original number given y.
(define y (+ x 3))
(define add3 (+ n 3))

But of course, to get back the number, you probably see that we could just easily add y with -3:

;; adding by -3 is the same as subtracting by 3
(define get-original-answer (+ y -3))

How about undoing a function?

(define example 10)
(define (sub3 x) (+ x -3))
(define (add3 x) (+ x 3))
(sub3 (add3 example))  ;; → 10
