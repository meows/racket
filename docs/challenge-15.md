# Challenge 15

Write a `gcd` function which can accept 2 or more naturals for input.

``` clojure
(gcd 8 12 16 24) ;; → 4
```

## Extra Challenge

Write an `lcm` function which can accept 2 or more naturals for input.

``` clojure
(lcm 8 12 16 24) ;; → 48
```
