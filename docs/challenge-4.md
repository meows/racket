# Racket Challenge #4

It's time to put everything you've done so far together into one puzzle. Give
the sum of all integers in `0..1000` that's divisible by either `3` or `5`.

## Possibly Clues

* `filter`
* `range`
* `and`
* `or`
* `if`
* `+`
* `apply`
