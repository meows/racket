Line

Define a function for each of these sequences, but before you do, identify (1) the initial starting point and (2) the velocity.

  • 0, 1, 2, 3, 4, 5...
  • 0, 2, 4, 6, 8...
  • 0, 3, 6, 9, 12...
  • 3, 6, 9, 12, 15...
  • -6, -1, 4, 9, 14...
  • 14, 9, 4, -1, -6...

Define a function in Racket called line which can take two parameters, starting position p0 and velocity v, and return a lambda which would give that sequence.

(define (line p0 v)
        (lambda (x) (+ p0 (* x v)))
)

(map (line 0 3) (range 10))
