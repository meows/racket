Problem

Find the sum of all even Fibonacci numbers whose values don't exceed 4 million.

Hint

;; finds the nth triangle number based on n
(define (triangle n)
    (if (zero? n)
        0
        (+ n (triangle (- n 1))))
)

;; finds the sum of all triangle numbers from n to 99
(define (challenge n sum)
    (if (< 100 (triangle n))
        sum
        (challenge (+ n 1) 
                   (+ sum (triangle n))))
)

(challenge 0 0) ;; → 455
