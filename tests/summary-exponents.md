# Exponentiation

Assume for the following that ${a, b, x}$ are reals.

## Multiplication

$a^x * a^y = a^{x + y}$

## Division

$\dfrac{a^x}{a^y} = a^{x - y}$

## Exponentiation

$(a^x)^y = a^{xy}$

## Negative Exponent

$a^{-1} = \dfrac{1}{a}$

$a * a^{-1} = 1$

$(\dfrac{a}{b})^{-n} = \dfrac{b^n}{a^n}$
